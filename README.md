## 📌 Project Overview

The **Bank Management System** is a console-based Python application that simulates real-world banking operations.

It allows users to perform core banking tasks like account creation, deposit, withdrawal, and balance checking through a simple menu-driven interface.

> 🎯 Goal: Understand Python fundamentals through a real-world mini project.

---
